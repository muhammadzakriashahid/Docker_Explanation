What is Docker?

Docker is a platform that enables developers to automate the deployment of applications inside lightweight, portable containers. Containers package an application and its dependencies together, ensuring consistent runs across different environments.

With Docker, you can:

  • Run applications in isolated environments
  • Share and distribute applications easily
  • Scale applications quickly and efficiently

Docker Syntax

Docker uses a specific syntax in its commands. Here are some common commands:

  • docker build: Builds an image from a Dockerfile
  • docker run: Runs a container from an image
  • docker push: Uploads an image to Docker Hub
  • docker pull: Downloads an image from Docker Hub
  • docker images: Lists all images on your local machine
  • docker ps: Lists all running containers

Creating a Dockerfile

A Dockerfile is a text file that contains instructions on how to build a Docker image. Here's a simple example:

FROM ubuntu:20.04
RUN apt-get update && apt-get install -y python3
COPY . /app
WORKDIR /app
CMD ["python3", "app.py"]

This Dockerfile does the following:

  • FROM: Specifies the base image (Ubuntu 20.04)
  • RUN: Executes commands in the container
  • COPY: Copies files from your local machine to the container
  • WORKDIR: Sets the working directory for subsequent commands
  • CMD: Specifies the command to run when the container starts

Building an Image

To build an image from your Dockerfile, use the following command:

docker build -t my-python-app .

This command builds the image and tags it as my-python-app. The . specifies the current directory as the build context.

Running a Container

Once the image is built, you can run a container using:

docker run -d -p 5000:5000 my-python-app

This command does the following:

  • -d: Runs the container in detached mode (background)
  • -p 5000:5000: Maps port 5000 of the container to port 5000 on your host machine
  • my-python-app: Specifies the image to run

After running this command, your application will be accessible at http://localhost:5000.

Uploading an Image to Docker Hub

To upload your image to Docker Hub, follow these steps:

  1. Log in to Docker Hub: 
    docker login
  2. Tag your image with your Docker Hub username: 
    docker tag my-python-app your-dockerhub-username/my-python-app
  3. Push the image to Docker Hub: 
    docker push your-dockerhub-username/my-python-app

Make sure to replace your-dockerhub-username with your actual Docker Hub username.
